# A-day-in-my-life
# 🌸 A Day In My Life — Cinematic Web Experience

An interactive storytelling website that visually presents **a complete day in my life** using modern web animations, smooth scrolling, and cinematic transitions.

---

## 🚀 Live Website

👉 https://Aarya1105.github.io/A-day-in-my-life/

---

## 🎬 Project Overview

This project is a **scroll-based animated website** where each section represents a moment of my daily routine:

🦋 Butterfly intro
🌅 Sunrise
⏰ Wake up
🍳 Breakfast
🏫 College
🍱 Lunch
🛵 Scooty ride
🇩🇪 German class
🌇 Sunset return home
💻 Coding time
❤️ Family time
🍽 Dinner
😴 Sleep

The website creates a **cinematic storytelling experience** similar to modern Apple-style product pages.

---

## ✨ Features

* Smooth scrolling animations
* Scroll-triggered image motion
* Background music autoplay
* Cinematic transitions (morning → day → evening → night)
* Animated butterfly intro
* Moving scooty animation
* Fully responsive layout
* Interactive storytelling UI

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* GSAP Animation Library
* ScrollTrigger
* Locomotive Scroll

## ⚙️ How to Run Locally

1. Clone repository

```
git clone https://github.com/Aarya1105/A-day-in-my-life.git
```

2. Open project folder

3. Open **index.html** in browser

(No installation required)

---

## 🌐 Deployment

The project is deployed using **GitHub Pages**.

Steps:

1. Push project to GitHub
2. Go to Repository → Settings
3. Open **Pages**
4. Select branch `main`
5. Save
